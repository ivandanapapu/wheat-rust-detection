# Project Summary

Title: Wheat Rust Detection: Binary Image Classifier

Objective: Build a binary CNN classifier to classify wheat leaf images as rust affected or healthy. The model uses data augmentation such as flipping, rotation, zoom and colour/contrast jitter. The project reports precision, recall, F1-score and especially investigates false-negative rate.

ML Technique: Binary Image Classification using Convolutional Neural Network.

Frontend: React + Vite modern UI.
Backend: FastAPI prediction API.
Deployment: Frontend on Vercel, backend on Render/Railway.
